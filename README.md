# Go-React-Blog

## Used Dependencies
- React
- Go
- Postgresql (go-pg)
- Gin Service
- GoDotEnv
- Zerolog
- Air(Live Host Reloading For Go)

## Install & run

### Front-end

First of all, you need to add this proxy into package.json file.

```bash
"proxy": "http://localhost:8080"
```
[<img src="assets/img1.png">]

```bash
cd/app
```

```bash
npm install
```

```bash
npm start
```
